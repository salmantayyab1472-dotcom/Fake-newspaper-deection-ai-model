🚀 Getting Started
1. Python Environment (Detection API)
Prerequisites: Python 3.10+

Install Dependencies:

bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
(Note: Deep learning libraries are large, download on a stable connection).

Run Inference (Example):

bash
python run_example.py
2. Dashboard (Visualization)
Prerequisites: Python 3.10+ (Flask)

Run Locally:

bash
python app.py
Open: http://localhost:5000

🛠️ Workflow
Upload Data: Enter a news headline and upload the corresponding newspaper image in the UI.
Detect: The backend processes text (Bi-LSTM) and visual features (MobileNetV2).
Analyze: The model combines features to generate a probability score.
Visualize: View the Real or Fake prediction and AI confidence percentage on the glassmorphism dashboard.
☁️ Deployment
Web App: Deploy the Flask application (app.py) to Vercel, Render, or Heroku.
Backend/Training: Run the training scripts inside src/ on a GPU-enabled machine (local PC or Google Colab).
